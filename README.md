## 🧪 Basic Troubleshooting

### The app will not open

- Right-click the file and choose Run as administrator
- Check whether Windows blocked the file
- Re-download the package from the link above
- Make sure the file finished downloading

### The page does not load

- Check that the app is running
- Refresh the browser
- Try another browser
- Confirm the local address is correct

### Accounts do not work

- Check the login data
- Make sure the account is still active
- Remove and add the account again
- Test with one account before you add more

### Requests fail or time out

- Lower the concurrency limit
- Check your proxy settings
- Confirm your internet connection
- Try again with one request

### The dashboard shows old data

- Refresh the page
- Restart the app
- Clear the browser cache
- Check whether telemetry or status polling is on
